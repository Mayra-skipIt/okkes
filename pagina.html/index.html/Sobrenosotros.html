¿Cómo creamos la pagina?

Creamos a pagina atraves de Visual Studio Code es un editor de código fuente desarrollado por Microsoft, esta disponible para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código entre otras cosas.
